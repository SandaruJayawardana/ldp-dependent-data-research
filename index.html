Local Differential Privacy Under Statistical Dependence

Information-theoretic privacy analysis, dependent data, and LDP mechanisms

About

My research focuses on privacy-preserving data collection under Local Differential Privacy (LDP), especially for multidimensional and statistically dependent user data.

Standard LDP mechanisms often assume that attributes are independent. However, real-world data frequently contains strong correlations. These dependencies may cause unintended information leakage, where observing the perturbed output of one attribute reveals information about another private attribute.

Research Themes

Correlation-Induced Leakage

Quantifying how dependencies between attributes amplify privacy leakage under LDP.

Dependency-Aware Privacy Metrics

Developing information-theoretic measures and bounds for privacy leakage in dependent data.

Coordinated Perturbation

Designing LDP mechanisms that use prior dependency knowledge to reduce unintended disclosure.

Privacy–Utility Trade-offs

Evaluating mechanisms using estimation error, leakage reduction, and attribute-inference attacks.

Projects

Dependency Triad

A framework for summarizing dependency structure and bounding correlation-induced privacy leakage.

Abstract

Dependency Triad studies privacy leakage in Local Differential Privacy when user attributes are statistically dependent. It provides a compact way to summarize dependency characteristics and estimate correlation-induced privacy leakage in multidimensional data.

Main Findings

  • Attribute dependencies can amplify privacy leakage under LDP.
  • Correlation-induced leakage can be analyzed using information-theoretic measures.
  • Compact dependency parameters can support efficient leakage estimation.

CoP: Coordinated Perturbation

A dependency-aware LDP mechanism for controlled disclosure under multidimensional data collection.

Abstract

CoP is a coordinated perturbation mechanism that uses prior dependency knowledge to reduce correlation-induced leakage while preserving useful disclosure for statistical analysis.

Main Findings

  • Coordinated perturbation can reduce unintended leakage.
  • Prior dependency knowledge can improve the privacy-utility trade-off.
  • CoP provides formal LDP protection while mitigating correlation-based inference.
